# **App Name**: CV Spark AI

## Core Features:

- User Authentication: Secure user registration, login, logout, and password recovery using Firebase Auth with email/password and Google login.
- CV Dashboard: Allows users to create, view, edit, and delete their CVs stored in Firestore.
- Real-time CV Editor: Section-based editor for personal information, profile, experience, education, skills, languages, and projects, dynamically updating the CV template.
- Template Management: Provides a selection of customizable CV templates, dynamically rendered with user data from Firestore (Modern, Professional, Minimalist).
- Real-time Preview: Provides a live preview of the CV as the user edits, ensuring an accurate representation of the final document.
- PDF Export: Generates and allows users to download their CV as a PDF file, both on the web and mobile platforms, leveraging Firebase Storage.
- AI-Powered Professional Profile Generator: AI tool that assists users in generating professional profile sections for their CVs using the Gemini API, to help those users unsure about wording their experience.

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) for a calming, trustworthy feel.
- Background color: Light gray (#F0F4F8), nearly white, providing a clean and modern canvas.
- Accent color: Muted Purple (#B39DDB) to add a touch of sophistication and highlight key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable experience.
- Modern design with cards, soft shadows, rounded borders, and a responsive layout adaptable to web and mobile platforms.
- Simple, professional icons to represent CV sections and actions, enhancing usability.
- Subtle animations for transitions and interactions to improve user engagement.
